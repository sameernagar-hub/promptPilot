# Prompt Engine

The first prompt engine uses rule-based logic for the completed MVP. The revised roadmap extends it into a profile-aware refinement engine before adding DSPy, retrieval, and broader evaluation.

Current MVP stages:

1. Classify the raw problem.
2. Generate clarifying questions.
3. Merge answers and tuning settings.
4. Produce multiple prompt variants.
5. Score and recommend the best option.

Next refinement stages:

1. Detect the domain and ask the user to confirm or correct it.
2. Inspect the user's prompting profile for relevant traits.
3. Ask clarifying questions before recommending a refined prompt.
4. Merge answers, profile preferences, and tuner settings.
5. Generate a detailed platform-aware prompt.
6. Explain which domain signals, answers, settings, and profile traits shaped the recommendation.

# Phase 7: Prompt Knowledge Base

## Goal

Use existing free/open prompt resources without becoming a copy-paste template library.

## Status

Not started.

## Source Rules

- Only ingest prompts that are open-source, user-submitted, explicitly licensed, public domain, permissively licensed, or generated by this project.
- Do not scrape paid or private prompt libraries.
- Track source and license metadata for every prompt source.

## Planned Pipeline

```txt
import source prompts
-> normalize fields
-> remove duplicates
-> classify/tag domain and intent
-> generate embeddings
-> store source/license metadata
-> make retrievable by semantic search
```

## Suggested Sources

- PromptSource by BigScience.
- Project-created prompt examples.
- User-submitted prompts with clear license or permission.
- Public domain or permissively licensed examples.

## Metadata To Track

- `source_name`
- `source_url`
- `license`
- `author`
- `allowed_usage`
- `domain`
- `intent`
- `prompt_type`
- `quality_score`

## Verification

- [ ] Every source has license metadata.
- [ ] Duplicate prompts are removed.
- [ ] Prompt source ingestion never copies from paid or private libraries.
- [ ] Retrieval stores patterns safely for later synthesis.
